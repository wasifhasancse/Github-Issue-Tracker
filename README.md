1. What is the difference between var, let, and const?
Answer:
var: It is a keyword to declare variables. It allow to changing value and redeclared.
let: It is the modern way to declare variables. It has block scope and the value can be changed but cannot be redeclared.
const: It is also block scoped, but its value cannot be changed and reassigned after declaration.

2. What is the spread operator (...)?
Answer:
It is used to expand or spread the elements of an array or object. It helps to copy arrays, merge arrays, or passing multiple values.

3. What is the difference between map(), filter(), and forEach()?
Answer:
map: It creates a new array by applying a function to every element.
filter: It creates a new array with the elements that match the given condition.
forEach: It runs a function for each element but it does not return a new array.
